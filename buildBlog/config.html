# 创建文件/文件夹

  1. 根目录下创建 docs 文件夹，vuePress中的绝大多数文件都将会包含在其中

  2. 创建 docs/.vuePress 文件夹，vuePress默认配置 及 默认主题 以及自定义组件 相关的内容都将会在此。

  3. 创建 .vuePress/public 文件夹，之后 config.js 访问的 静态资源文件的 路径 '/' 指向的 就是 该文件夹。

  4. 创建 .vuePress/config.js文件，该文件为默认配置，包含nav、sidebar、public等的路由配置。

  5. 创建默认主题首页docs/README.md 文件（如果你需要自定义首页，请访问 自定义主题。

# 配置config.js

module.exports = {
    title: 'V 的博客',    //首页的标题
    description: 'V的博客',  //描述
    head: [
        ['link', { rel: 'icon', href: '/favicon.ico' }],  //public 中的资源
        ['link', { rel: 'manifest', href: '/manifest.json' }], //public 中的相关资源
    ],
    themeConfig:{
        nav: [   //导航栏
            { text: '主页', link: '/' },   
            { text: '教程',  // 下拉菜单导航
                items: [
                    { text: '如何搭建此Blog', link: '/buildBlog/' },
                    { text: 'JavaScript', link: '/lessones/JavaScript/' },
                    { text: 'Css', link: '/lessones/Css/' },
                    { text: 'App', link: '/lessones/App/' },
                    { text: '小程序', link: '/lessones/小程序/' }
                ] 
            },
            { text: '随笔', link: '/essay/' },  
            { text: 'Bug', link: '/bug/' },
            { text: '关于', link: '/about/' },
            { text: 'Github', link: 'https://www.github.com/DaskV' },
        ],
        sidebar:{  //侧边栏 路由配置 （请在docs目录下创建 相关文件夹）
            '/buildBlog/':[
                {
                    title: 'VuePress 快速搭建Blog', // 侧边栏名称          
                    collapsable: false, // 不可折叠       
                    children: [     
                        ['','介绍'],     // '':代表该文件夹(buildBlog)下 默认的README.md 里的 内容, 后面的 文字 代表 别名
                        ['start','快速上手'],  //'start该文件夹(buildBlog)下 start.md' , 可以不写 .md 后缀
                        ['config','关键配置']
                    ]
                }         
            ],

        },
        sidebarDepth: 1,   //测试兰 展开深度
        lastUpdated: 'Last Updated',   //最后更新时间
        serviceWorker: true,  
    }
}

提醒

vuePress中，默认主题情况下，所有的内容均以markdown文件呈现，且必须有一个README.md 文件 作为 默认展示页面

详细内容请访问官网默认主题配置

# 首页配置

打开.vuePress 下的 README.md , 输入以下内容

---
home: true                 // true 代表使用默认主题，如需使用自定义主题，请访问官网
heroImage: /portrait.png   //public 下的静态资源文件， 这里 为主页设置 一个 默认图片
actionText: 关于我 →
actionLink: /about/
features:
- title: 教程
  details: 与前端相关的一系列教程，用于学习、复习、面试等。
- title: 随笔
  details: 开发学习过程中遇到的问题，及解决方案，以及一些有意思的app,插件,动画等。
- title: Bug
  details: 各种奇奇怪怪的Bug，Bug killer的福地， 期待各位解决方案。
footer: MIT Licensed | Copyright © 2018-present DaskV
---

至此最简单的vuePress Blog 已经搭建完毕了。输入

npm run docs:dev
进行预览吧
